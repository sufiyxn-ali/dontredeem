       val output = module.forward(IValue.from(tokens))
       val scores = output.toTensor().dataAsFloatArray
       return if (scores[1] > 0.5) "SCAM" else "LEGITIMATE"
   }

🎓 TECHNICAL DETAILS
═════════════════════════════════════════════════════════════════════════

Architecture:
  Bidirectional LSTM processes text in both directions:
  
  Forward:  "Your account has been suspended"
            → Captures left context
  
  Backward: "suspended been has account Your"
            → Captures right context
  
  Result:   Full bidirectional understanding
            → Better scam detection accuracy

Attention Mechanism:
  • 4 attention heads
  • Focus on important tokens
  • "suspended" gets high attention (scam indicator)
  • "verified" gets low attention (legitimate)

Special Tokens:
  Custom vocabulary markers for scam detection:
  "[URGENT]" when sees: urgent, immediately, now, asap
  "[THREAT]" when sees: legal, suspended, warrant, lawsuit
  → Model learns these patterns matter for scam prediction
